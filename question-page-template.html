Question #1

Medium

Let $f: \mathbb{R} \to \mathbb{R}$ be a differentiable function such that $f'(x) = f(x)$ for all $x \in \mathbb{R}$ and $f(0) = 1$. What is the value of $f(2)$?

(A) $e$
(B) $e^2$
(C) $2e$
(D) $e^{1/2}$
(E) $2$

Solution:

The differential equation $f'(x) = f(x)$ with initial condition $f(0) = 1$ has the unique solution $f(x) = e^x$.

Therefore, $f(2) = e^2$.

Answer: (B) $e^2$

Question #2

Hard

Find the value of the integral:

$\int_0^{\pi/2} \frac{\sin x}{\sin x + \cos x} dx$

(A) $\frac{\pi}{4}$
(B) $\frac{\pi}{2}$
(C) $\frac{\pi}{3}$
(D) $\ln 2$
(E) $1$

Solution:

Let $I = \int_0^{\pi/2} \frac{\sin x}{\sin x + \cos x} dx$

We can make the substitution $u = \frac{\pi}{2} - x$ to get:

$I = \int_0^{\pi/2} \frac{\sin(\frac{\pi}{2} - u)}{\sin(\frac{\pi}{2} - u) + \cos(\frac{\pi}{2} - u)} du$

Which simplifies to:

$I = \int_0^{\pi/2} \frac{\cos u}{\cos u + \sin u} du$

Adding these two equations:

$2I = \int_0^{\pi/2} \frac{\sin x + \cos u}{\sin x + \cos x} dx = \int_0^{\pi/2} 1 dx = \frac{\pi}{2}$

Therefore $I = \frac{\pi}{4}$

Answer: (A) $\frac{\pi}{4}$
