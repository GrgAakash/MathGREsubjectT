About This Site

This is a collection of practice questions for the GRE Mathematics Subject Test, organized by topic with explanations whenever possible.

GRE Math Subject Test Coverage

The test covers these main areas:

  • Calculus - About 50% of the test
  • Abstract Algebra - About 25% of the test
  • Real Analysis - About 25% of the test
  • Other topics - Linear algebra, topology, complex analysis

For more information, visit the official GRE Math Subject Test fact sheet: https://www.ets.org/pdfs/gre/fact-sheet-math.pdf

It is important to be familiar with the fact-sheet to strategically prepare for the test.The Calculus section also covers materials from Multivariable Calculus like Green's theorem, partial derivatives and ODE.

How to Use

Each section contains practice questions with multiple choice answers. Click on an answer to see if it's correct, and use the "Show/Hide Explanation" link to see the detailed solution.

The questions are organized by topic, so you can focus on specific areas where you need more practice.

Test Format

The actual GRE Math Subject Test has about 66 multiple-choice questions to complete in 2 hours and 50 minutes.
