Limits Practice Questions

1. Evaluate \( \lim_{x\to 0} \frac{\cos(3x)-1}{x^2} \)

Source: (1),Practice 1, Q1

\( \frac{9}{2} \)
\( \frac{3}{2} \)
\( -\frac{2}{3} \)
\( -\frac{3}{2} \)
\( -\frac{9}{2} \)

Show/Hide Explanation

Use the limit identity: \( \lim_{x \to 0} \frac{\cos x - 1}{x^2} = -\frac{1}{2} \). Let \( u = 3x \), then the limit becomes \( 9 \cdot (-\frac{1}{2}) = -\frac{9}{2} \).

Answer: (E) \( -\frac{9}{2} \)

2. Let \( g(x) = e^{2x + 1} \). Evaluate \( \lim_{x \to 0} \frac{g(g(x)) - g(e)}{x} \)

Source: (1),Practice 1, Q20

\( 2e \)
\( 4e \)
\( e \)
\( 2e^{2e+1} \)
\( 4e^{2e+2} \)

Show/Hide Explanation

Solution:

  1. First compute \( g(g(x)) \): \[ g(g(x)) = e^{2e^{2x+1} + 1} \]
  2. At \( x = 0 \): \[ g(e) = e^{2e + 1} \]
  3. The limit is a 0/0 form, so apply L'Hôpital's Rule: \[ \frac{d}{dx}[g(g(x))] = g'(g(x)) \cdot g'(x) \] \[ g'(x) = 2e^{2x+1} \]
  4. Evaluate at \( x = 0 \): \[ g'(0) = 2e \] \[ g'(e) = 2e^{2e+1} \] \[ \text{Limit} = g'(e) \cdot g'(0) = 4e^{2e+2} \]

Answer: (E) \( 4e^{2e+2} \)

3. Evaluate \( \lim_{x \to 1^-} \frac{x - 1}{|x - 1|} \)

Source: (2),Page 39

\( -1 \)
\( 1 \)
\( \frac{1}{e} \)
\( -\frac{1}{e} \)
undefined

Show/Hide Explanation

For \( x < 1 \), \( |x - 1| = 1 - x \), so the expression becomes \( \frac{x - 1}{1 - x} = -1 \).

Answer: (A) \( -1 \)

4. Let \( \varphi(x) = \int_0^x \varphi(t)\,dt + \sin x \). Evaluate \( \lim_{x\to\pi/2} \left( 2\varphi(x) - e^x \right) \)

Source: (3), 2025, Q29

1
2
3
4

Show/Hide Explanation

Differentiate both sides: \( \varphi' = \varphi + \cos x \). Solve ODE to get \( \varphi(x) = \frac{\sin x - \cos x}{2} + \frac{e^x}{2} \). Then \( 2\varphi(x) - e^x = \sin x - \cos x \), so at \( x = \frac{\pi}{2} \), we get \( 1 - 0 = 1 \).

Answer: (A) 1

Continuity Practice Questions

1. Is the function \(f(x) = \frac{x^2 - 4}{x - 2}\) continuous at \(x = 2\)?

A) Yes
B) No

Show/Hide Explanation

The function is undefined at \(x = 2\) because the denominator is zero. Therefore, it is not continuous at \(x = 2\).

Answer: B) No

2. What value of \(k\) makes the function \(f(x) = \begin{cases} 2x + 1 & x \neq 1 \\ k & x = 1 \end{cases}\) continuous at \(x = 1\)?

A) 1
B) 2
C) 3
D) 4
E) Undefined

Show/Hide Explanation

For continuity at \(x = 1\), we need \(\lim_{x \to 1} f(x) = f(1)\). \(\lim_{x \to 1} (2x + 1) = 2(1) + 1 = 3\). \(f(1) = k\). Thus, we need \(k = 3\).

Answer: C) 3

3. Is the function \(f(x) = \sqrt{x - 4}\) continuous at \(x = 4\)?

A) Yes
B) No

Show/Hide Explanation

The square root function is continuous on its domain. The domain of \(\sqrt{x - 4}\) is \(x \geq 4\). Since \(x=4\) is in the domain, the function is continuous at \(x = 4\).

Answer: A) Yes

4. Is the function \(f(x) = \frac{1}{x - 5}\) continuous at \(x = 5\)?

A) Yes
B) No

Show/Hide Explanation

The function is undefined at \(x = 5\) because the denominator is zero. Therefore, it is not continuous at \(x = 5\).

Answer: B) No

5. Is the function \(f(x) = \cos(x)\) continuous for all real numbers?

A) Yes
B) No

Show/Hide Explanation

The cosine function is a fundamental trigonometric function and is continuous for all real numbers.

Answer: A) Yes

6. Is the function \(f(x) = \frac{x}{|x|}\) continuous at \(x = 0\)?

A) Yes
B) No

Show/Hide Explanation

The function is undefined at \(x = 0\) because the denominator is zero. Therefore, it is not continuous at \(x = 0\).

Answer: B) No

7. Is the function \(f(x) = e^{x}\) continuous for all real numbers?

A) Yes
B) No

Show/Hide Explanation

The exponential function \(e^x\) is continuous for all real numbers.

Answer: A) Yes

8. Is the function \(f(x) = \frac{1}{x}\) continuous at \(x = 0\)?

A) Yes
B) No

Show/Hide Explanation

The function is undefined at \(x = 0\) because the denominator is zero. Therefore, it is not continuous at \(x = 0\).

Answer: B) No

9. Is the function \(f(x) = x^2 + 2x\) continuous for all real numbers?

A) Yes
B) No

Show/Hide Explanation

Polynomial functions, including \(x^2 + 2x\), are continuous for all real numbers.

Answer: A) Yes

10. Is the function \(f(x) = \tan(x)\) continuous at \(x = \frac{\pi}{2}\)?

A) Yes
B) No

Show/Hide Explanation

The tangent function is undefined at \(x = \frac{\pi}{2}\) because \(\cos\left(\frac{\pi}{2}\right) = 0\). Therefore, it is not continuous at \(x = \frac{\pi}{2}\).

Answer: B) No

Differentiation Practice Questions

1. Find the derivative of \(f(x) = x^3 - 4x^2 + 5x - 7\).

A) 3x^2 - 4x + 5
B) 3x^2 - 8x + 5
C) x^2 - 8x + 5
D) 3x^3 - 8x + 5
E) 3x^2 - 8x - 7

Show/Hide Explanation

Using the power rule, \(\frac{d}{dx}(x^n) = n x^{n-1}\): \(f'(x) = 3x^{3-1} - 4(2x^{2-1}) + 5(1x^{1-1}) - 0 = 3x^2 - 8x + 5\).

Answer: B) \(3x^2 - 8x + 5\)

2. What is the derivative of \(g(x) = \sin(5x)\)?

A) \cos(5x)
B) 5\sin(5x)
C) 5\cos(5x)
D) -\cos(5x)
E) \frac{1}{5}\cos(5x)

Show/Hide Explanation

Using the chain rule, \(\frac{d}{dx}(\sin(u)) = \cos(u) \cdot \frac{du}{dx}\). Here \(u = 5x\), so \(\frac{du}{dx} = 5\). \(g'(x) = \cos(5x) \cdot 5 = 5\cos(5x)\).

Answer: C) \(5\cos(5x)\)

3. Find the derivative of \(h(x) = e^{x^2}\).

A) e^{x^2}
B) 2x e^{x^2}
C) x e^{x^2}
D) e^{2x}
E) e^{x^2+1}

Show/Hide Explanation

Using the chain rule, \(\frac{d}{dx}(e^u) = e^u \cdot \frac{du}{dx}\). Here \(u = x^2\), so \(\frac{du}{dx} = 2x\). \(h'(x) = e^{x^2} \cdot 2x = 2x e^{x^2}\).

Answer: B) \(2x e^{x^2}\)

4. Find the derivative of \(k(x) = \ln(x^2 + 1)\).

A) \frac{2x}{x^2 + 1}
B) \frac{1}{x^2 + 1}
C) \frac{1}{x}
D) 2x \ln(x^2 + 1)
E) \frac{x^2 + 1}{x}

Show/Hide Explanation

Using the chain rule, \(\frac{d}{dx}(\ln(u)) = \frac{1}{u} \cdot \frac{du}{dx}\). Here \(u = x^2 + 1\), so \(\frac{du}{dx} = 2x\). \(k'(x) = \frac{1}{x^2 + 1} \cdot 2x = \frac{2x}{x^2 + 1}\).

Answer: A) \(\frac{2x}{x^2 + 1}\)

5. Find the derivative of \(m(x) = \frac{x}{x + 1}\).

A) \frac{1}{(x + 1)^2}
B) \frac{-1}{(x + 1)^2}
C) \frac{2x + 1}{(x + 1)^2}
D) \frac{1}{(x + 1)^2}
E) \frac{x}{(x + 1)^2}

Show/Hide Explanation

Using the quotient rule, \(\frac{d}{dx}\left(\frac{u}{v}\right) = \frac{u' v - u v'}{v^2}\). Here \(u = x\), \(v = x + 1\). \(u' = 1\), \(v' = 1\). \(m'(x) = \frac{1 \cdot (x + 1) - x \cdot 1}{(x + 1)^2} = \frac{x + 1 - x}{(x + 1)^2} = \frac{1}{(x + 1)^2}\).

Answer: D) \(\frac{1}{(x + 1)^2}\)

6. What is the derivative of \(p(x) = \tan(x^2)\)?

A) \sec^2(x^2)
B) 2x \sec^2(x^2)
C) \sec(x^2) \tan(x^2)
D) 2x \sec(x^2) \tan(x^2)
E) \frac{1}{\cos^2(x^2)}

Show/Hide Explanation

Using the chain rule, \(\frac{d}{dx}(\tan(u)) = \sec^2(u) \cdot \frac{du}{dx}\). Here \(u = x^2\), so \(\frac{du}{dx} = 2x\). \(p'(x) = \sec^2(x^2) \cdot 2x = 2x \sec^2(x^2)\).

Answer: B) \(2x \sec^2(x^2)\)

7. Find the derivative of \(q(x) = \sqrt{x^3 + 1}\).

A) \frac{3x^2}{\sqrt{x^3 + 1}}
B) \frac{1}{2\sqrt{x^3 + 1}}
C) \frac{3x^2}{2\sqrt{x^3 + 1}}
D) \frac{2}{3\sqrt{x^3 + 1}}
E) 3x^2 \sqrt{x^3 + 1}

Show/Hide Explanation

Rewrite as \(q(x) = (x^3 + 1)^{1/2}\). Using the chain rule, \(\frac{d}{dx}(u^n) = n u^{n-1} \cdot \frac{du}{dx}\). Here \(u = x^3 + 1\), so \(\frac{du}{dx} = 3x^2\). \(q'(x) = \frac{1}{2} (x^3 + 1)^{-1/2} \cdot 3x^2 = \frac{3x^2}{2 \sqrt{x^3 + 1}}\).

Answer: C) \(\frac{3x^2}{2 \sqrt{x^3 + 1}}\)

8. What is the derivative of \(r(x) = x \cos(x)\)?

A) \sin(x)
B) \cos(x)
C) \cos(x) + x \sin(x)
D) \cos(x) - x \sin(x)
E) \sin(x) - \cos(x)

Show/Hide Explanation

Using the product rule, \(\frac{d}{dx}(u v) = u' v + u v'\). Here \(u = x\), \(v = \cos(x)\). \(u' = 1\), \(v' = -\sin(x)\). \(r'(x) = 1 \cdot \cos(x) + x \cdot (-\sin(x)) = \cos(x) - x \sin(x)\).

Answer: D) \(\cos(x) - x \sin(x)\)

9. Find the derivative of \(s(x) = \frac{\ln(x)}{x}\).

A) \frac{1 - \ln(x)}{x^2}
B) \frac{1 + \ln(x)}{x^2}
C) \frac{1}{x} - 1
D) \frac{1}{x}
E) \ln(x) - 1

Show/Hide Explanation

Using the quotient rule, \(\frac{d}{dx}\left(\frac{u}{v}\right) = \frac{u' v - u v'}{v^2}\). Here \(u = \ln(x)\), \(v = x\). \(u' = \frac{1}{x}\), \(v' = 1\). \(s'(x) = \frac{\frac{1}{x} \cdot x - \ln(x) \cdot 1}{x^2} = \frac{1 - \ln(x)}{x^2}\).

Answer: A) \(\frac{1 - \ln(x)}{x^2}\)

10. What is the derivative of \(t(x) = e^{-2x}\)?

A) e^{-2x}
B) 2 e^{-2x}
C) -2 e^{-2x}
D) -\frac{1}{2} e^{-2x}
E) -e^{-2x}

Show/Hide Explanation

Using the chain rule, \(\frac{d}{dx}(e^u) = e^u \cdot \frac{du}{dx}\). Here \(u = -2x\), so \(\frac{du}{dx} = -2\). \(t'(x) = e^{-2x} \cdot (-2) = -2 e^{-2x}\).

Answer: C) \(-2 e^{-2x}\)
