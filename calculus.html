Question #1

Easy

If $f(x) = x^3 \sin(x)$, then $f'(\pi)$ equals:

(A) $0$
(B) $\pi^3$
(C) $-\pi^3$
(D) $3\pi^2$
(E) $-3\pi^2$

Solution:

Using the product rule to find $f'(x)$:

$f'(x) = \frac{d}{dx}[x^3 \sin(x)]$

$f'(x) = 3x^2 \sin(x) + x^3 \cos(x)$

Now we evaluate at $x = \pi$:

$f'(\pi) = 3\pi^2 \sin(\pi) + \pi^3 \cos(\pi)$

Since $\sin(\pi) = 0$ and $\cos(\pi) = -1$, we get:

$f'(\pi) = 3\pi^2 \cdot 0 + \pi^3 \cdot (-1) = -\pi^3$

Answer: (C) $-\pi^3$

Question #2

Medium

Evaluate the definite integral:

$\int_0^{\pi/4} \sec^2(x) \tan(x) \, dx$

(A) $\frac{1}{2}$
(B) $1$
(C) $\frac{1}{\sqrt{2}}$
(D) $\frac{\pi}{4}$
(E) $\frac{1}{3}(\sec^2(\frac{\pi}{4}) - 1)$

Solution:

Let's use substitution. Let $u = \tan(x)$, which gives us $du = \sec^2(x) \, dx$.

$\int_0^{\pi/4} \sec^2(x) \tan(x) \, dx = \int_0^{\pi/4} \tan(x) \, du = \int_0^1 u \, du$

We changed the bounds because when $x = 0$, $u = \tan(0) = 0$, and when $x = \pi/4$, $u = \tan(\pi/4) = 1$.

$\int_0^1 u \, du = \left. \frac{u^2}{2} \right|_0^1 = \frac{1^2}{2} - \frac{0^2}{2} = \frac{1}{2}$

Answer: (A) $\frac{1}{2}$

Question #3

Medium

For the power series $\sum_{n=0}^{\infty} \frac{x^n}{n!}$, which of the following statements is true?

(A) The series converges only for $x = 0$
(B) The series converges for $|x| < 1$
(C) The series converges for all real values of $x$
(D) The series converges for $x \geq 0$
(E) The series diverges for all values of $x$

Solution:

The power series $\sum_{n=0}^{\infty} \frac{x^n}{n!}$ is the Taylor series expansion of $e^x$.

To determine the radius of convergence, we can use the ratio test:

$\lim_{n \to \infty} \left| \frac{a_{n+1}}{a_n} \right| = \lim_{n \to \infty} \left| \frac{\frac{x^{n+1}}{(n+1)!}}{\frac{x^n}{n!}} \right| = \lim_{n \to \infty} \left| \frac{x^{n+1} \cdot n!}{(n+1)! \cdot x^n} \right| = \lim_{n \to \infty} \left| \frac{x}{n+1} \right| = 0$

Since this limit is 0 for any value of $x$, the series converges for all real values of $x$.

Answer: (C) The series converges for all real values of $x$

Question #4

Hard

Let $f(x,y) = x^2y + xy^2$ and $C$ be the curve parametrized by $\mathbf{r}(t) = (t^2, t^3)$ for $0 \leq t \leq 1$. What is the value of the line integral $\int_C \nabla f \cdot d\mathbf{r}$?

(A) $1$
(B) $2$
(C) $3$
(D) $f(1,1) - f(0,0)$
(E) $0$

Solution:

The gradient $\nabla f$ is given by:

$\nabla f = \left( \frac{\partial f}{\partial x}, \frac{\partial f}{\partial y} \right) = (2xy + y^2, x^2 + 2xy)$

For a line integral of the form $\int_C \nabla f \cdot d\mathbf{r}$, when $f$ is a scalar field, this is equivalent to evaluating $f$ at the endpoints of the curve. This is the fundamental theorem of calculus for line integrals.

$\int_C \nabla f \cdot d\mathbf{r} = f(\mathbf{r}(1)) - f(\mathbf{r}(0))$

$\mathbf{r}(0) = (0, 0)$ and $\mathbf{r}(1) = (1, 1)$

$f(0,0) = 0^2 \cdot 0 + 0 \cdot 0^2 = 0$

$f(1,1) = 1^2 \cdot 1 + 1 \cdot 1^2 = 1 + 1 = 2$

$\int_C \nabla f \cdot d\mathbf{r} = f(1,1) - f(0,0) = 2 - 0 = 2$

Answer: (B) $2$

Question #5

Hard

Evaluate the limit:

$\lim_{n \to \infty} \left(1 + \frac{3}{n}\right)^{2n}$

(A) $e^3$
(B) $e^6$
(C) $1$
(D) $\infty$
(E) $3$

Solution:

We can use the well-known limit $\lim_{n \to \infty} \left(1 + \frac{a}{n}\right)^n = e^a$.

In our case, we have:

$\lim_{n \to \infty} \left(1 + \frac{3}{n}\right)^{2n}$

Let's rewrite this as:

$\lim_{n \to \infty} \left[\left(1 + \frac{3}{n}\right)^{n}\right]^2$

$= \left[\lim_{n \to \infty} \left(1 + \frac{3}{n}\right)^{n}\right]^2$

$= (e^3)^2 = e^6$

Answer: (B) $e^6$

Question #6

Medium

Consider the differential equation $\frac{dy}{dt} + 2y = 4\sin(t)$ with initial condition $y(0) = 1$. What is the value of $y(\pi)$?

(A) $1$
(B) $e^{-2\pi}$
(C) $e^{-2\pi} + 1$
(D) $e^{-2\pi} + 2$
(E) $\frac{4}{5}e^{-2\pi} + \frac{4}{5}$

Solution:

This is a first-order linear differential equation. We'll use the integrating factor method.

The integrating factor is $e^{\int 2 dt} = e^{2t}$.

Multiplying both sides of the equation by $e^{2t}$:

$e^{2t}\frac{dy}{dt} + 2e^{2t}y = 4e^{2t}\sin(t)$

This equals: $\frac{d}{dt}(e^{2t}y) = 4e^{2t}\sin(t)$

Integrating both sides:

$e^{2t}y = \int 4e^{2t}\sin(t) dt + C$

Using integration by parts to evaluate $\int e^{2t}\sin(t) dt$:

Let $u = \sin(t)$ and $dv = e^{2t} dt$, then $du = \cos(t) dt$ and $v = \frac{1}{2}e^{2t}$.

$\int e^{2t}\sin(t) dt = \frac{1}{2}e^{2t}\sin(t) - \int \frac{1}{2}e^{2t}\cos(t) dt$

Applying integration by parts again on the second term:

Let $u = \cos(t)$ and $dv = e^{2t} dt$, then $du = -\sin(t) dt$ and $v = \frac{1}{2}e^{2t}$.

$\int \frac{1}{2}e^{2t}\cos(t) dt = \frac{1}{4}e^{2t}\cos(t) + \int \frac{1}{4}e^{2t}\sin(t) dt$

Combining and solving for $\int e^{2t}\sin(t) dt$:

$\int e^{2t}\sin(t) dt = \frac{1}{2}e^{2t}\sin(t) - \frac{1}{4}e^{2t}\cos(t) - \int \frac{1}{4}e^{2t}\sin(t) dt$

$\frac{5}{4}\int e^{2t}\sin(t) dt = \frac{1}{2}e^{2t}\sin(t) - \frac{1}{4}e^{2t}\cos(t)$

$\int e^{2t}\sin(t) dt = \frac{2}{5}e^{2t}\sin(t) - \frac{1}{5}e^{2t}\cos(t) + C$

Therefore:

$e^{2t}y = 4[\frac{2}{5}e^{2t}\sin(t) - \frac{1}{5}e^{2t}\cos(t)] + C$

$e^{2t}y = \frac{8}{5}e^{2t}\sin(t) - \frac{4}{5}e^{2t}\cos(t) + C$

$y = \frac{8}{5}\sin(t) - \frac{4}{5}\cos(t) + Ce^{-2t}$

Using the initial condition $y(0) = 1$:

$1 = \frac{8}{5} \cdot 0 - \frac{4}{5} \cdot 1 + C$

$1 = -\frac{4}{5} + C$

$C = 1 + \frac{4}{5} = \frac{9}{5}$

Thus, the solution is:

$y(t) = \frac{8}{5}\sin(t) - \frac{4}{5}\cos(t) + \frac{9}{5}e^{-2t}$

Evaluating at $t = \pi$:

$y(\pi) = \frac{8}{5}\sin(\pi) - \frac{4}{5}\cos(\pi) + \frac{9}{5}e^{-2\pi}$

$y(\pi) = \frac{8}{5} \cdot 0 - \frac{4}{5} \cdot (-1) + \frac{9}{5}e^{-2\pi}$

$y(\pi) = \frac{4}{5} + \frac{9}{5}e^{-2\pi}$

This doesn't match any of the given options exactly. The closest is option (E), but the coefficient of $e^{-2\pi}$ should be $\frac{9}{5}$ not $\frac{4}{5}$. Since this is a sample problem, there may be an error in the options.

Answer: The correct expression is $\frac{4}{5} + \frac{9}{5}e^{-2\pi}$, which doesn't match exactly with any option. Option (E) is closest.
